## Installation and basic usage
[![asciicast](https://asciinema.org/a/276023.svg)](https://asciinema.org/a/276023)

Installation using npm:
```
$ npm i -g npm
$ npm i another-one-brain-games
```

Running games:
```
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
```
